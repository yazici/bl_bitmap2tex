bitmap2tex is a blender addon.  
use blender as pbr texture genarater.

## usage
- install addon
- save enable addon
- enable bitmap2tex and [Node Wrangler](https://gregzaal.github.io/node-wrangler/)

workflow
- init project
- save blend file
- edit bitmap image
- adjust comp
- render
- adjust
